# Aprendix_EquipoG__
Esta es una una herramienta tecnológica divertida, animada con jugabilidad para mostrar las principales metodologías para administrar proyectos y que tenga retroalimentación de lo aprendido, y sugerencias para casos de uso y aprendizaje en corto tiempo que pueda ser accesada desde cualquier lugar.

¿Qué vamos a hacer?
R= Como equipo, vamos a hacer una aplicación que funcione con y sin internet que permita al usuario visualizar las principales metodologías para administrar proyectos.
¿Qué otras funciones debe tener la aplicación?
R= Retroalimentación de lo aprendido en forma de evaluación al final de cada lección y metodología, además de tips y consejos, en una pestaña extra y entre lecciones.
¿Cómo lo haremos?
R= Una página web que puede ser accesada desde cualquier lugar del mundo, siempre y cuando se cumplan condiciones de internet y red. Para este proyecto haremos una página web que se comporte como una applicación de escritorio o móvil. Instalándose como si fuera una aplicación funcionando offline (con caché), usando https obligatorio.
Gráficos, para que el usuario pueda interactuar con la aplicación
R= Necesitaremos una librería gráfica que se encarga de renderizar y manejar la interfaz visual. Un ejemplo sería SFML (Simple Fast Multimedia Library) que proporciona una interfaz simple a varios componentes de la PC, usada para videojuegos y aplicaciones multimedia.
¿En caso de necesitar “base de datos”?
R= En situaciones de soporte técnico con la herramienta, implementaremos una conexión a base de datos con la aplicación, utilizando mySQL por su Licencia gratuita.


Información sobre las metodologías de Administración de Proyectos
Para la primera versión del proyecto, obtendremos la información de fuentes confiables como Pmbook versión 5, o algún otro. En caso de tener que actualizar o añadir más información o metodologías nuevas, proporcionar al usuario opción de actualizar la aplicación.
